# Field classification

This document records how fields from the Reading List fixtures are classified
into bibliographic, workflow, and task-related fields. It is a living
specification used by the importer and mapping rules.

Sections:
- Bibliographic fields (title, authors, doi, journal, url, zotero_key)
- Workflow fields (`Status`, `Status_1`)
- Task-related fields (tables and summary tables)

Each section should contain expected types and normalization rules.
